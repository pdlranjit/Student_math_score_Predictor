#Student Math Score Predictor🎓

#Overview
Predicts student math scores using Linear Regression.

#Features
- Trained on Students Performance dataset
- REST API built with FastAPI
- Input: gender, race, education, lunch, test prep

#How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run API: uvicorn main:app --reload
3. Visit: http://127.0.0.1:8000/docs
